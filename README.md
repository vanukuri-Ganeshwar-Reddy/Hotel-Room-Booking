# 🏨 Hotel Room Booking System

## Project Description
This is a Python-based Hotel Room Booking System. It allows users to book rooms, view bookings, search bookings, cancel bookings, and generate bills through a simple console application.

## Features
- Book a room
- View bookings
- Search bookings
- Cancel bookings
- Generate bill

## Technologies Used
- Python
- Functions
- Modules
- File Handling

## How to Run
1. Download or clone this repository.
2. Open the project in VS Code.
3. Run:

```bash
python main.py
```
